# BioRoute - Sistema de Gestión Logística y Trazabilidad

Proyecto BioRoute - TP1
* App.java: Código fuente del prototipo operacional en Java.
* script_db.sql: Script de creación de tablas y carga de datos iniciales en MySQL.
* Instrucciones: Para ejecutarlo, es necesario cargar el driver conector de MySQL en las librerías del proyecto.
  
### Descripción
Este proyecto es un prototipo operacional desarrollado para la Actividad Práctica 1. El sistema permite gestionar la trazabilidad de pedidos de alimentos orgánicos.

### Componentes Técnicos
* **Lenguaje:** Java 17.
* **IDE:** Visual Studio Code.
* **Base de Datos:** MySQL 8.0.
* **Conectividad:** JDBC (Java Database Connectivity).

### Estructura del Repositorio
* `App.java`: Contiene la lógica de conexión y consulta a la base de datos.
* `script_db.sql`: Script para la creación de la base de datos `bioroute_db` y tablas de Clientes, Productos y Pedidos.
